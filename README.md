# CareerOS

AI-powered career growth and professional networking platform built on ServiceNow.

## Features

- Career Profiles
- Career Roles
- Certifications
- Skills Management
- Role Skill Mapping
- Profile Skill Tracking
- Skill Gap Analysis
- Career Score Engine
- GitHub Source Control Integration

## Tech Stack

- ServiceNow App Engine Studio
- GlideRecord
- Business Rules
- Script Includes
- Client Scripts
- UI Policies
- GitHub Source Control
